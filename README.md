# Memecoin Launchpad

This repository contains two projects:

## 1. Meme Launchpad

A Next.js application for launching and managing meme tokens on the blockchain.

```
cd meme-launchpad
pnpm install
pnpm run dev
```

## 2. Eliza Mantle Bot

A Twitter bot for the Mantle Network built with Node.js.

```
cd eliza-mantle-bot
npm install
npm run start
```
